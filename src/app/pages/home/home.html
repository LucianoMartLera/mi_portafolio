Inicio

Soy Lucas Luciano Martínez Lera, desarrollador web con base en Salta, Argentina. Me dedico a construir sitios web modernos, funcionales y visualmente atractivos, combinando diseño cuidado con código limpio y eficiente. Si tenés un proyecto en mente, estoy listo para hacerlo realidad.

A qué me dedico

Me especializo en el desarrollo web, abarcando tanto la parte visual como la lógica detrás de cada sitio. Trabajo con HTML y CSS para maquetar interfaces responsivas y estéticamente cuidadas, uso JavaScript para agregar interactividad y dinamismo, y aplico Python para resolver la lógica de backend y automatización de procesos. Me enfoco en crear productos digitales que sean intuitivos para el usuario y sólidos por dentro.

Experiencia

Si bien me encuentro en una etapa de crecimiento profesional activo, he desarrollado una base técnica sólida a través de proyectos personales y trabajos académicos. Cada proyecto que encaré fue una oportunidad para aprender algo nuevo y mejorar mi forma de escribir código. Me considero una persona autodidacta, con capacidad para adaptarme rápidamente a nuevas tecnologías y entornos de trabajo. Actualmente me encuentro profundizando mis conocimientos en Angular para el desarrollo de aplicaciones web más robustas y escalables.
